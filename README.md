# OCT-Retinal-Anomaly-Detector

Analyzed real OCT scan data from Calcutta Lions Netra Niketan for post-trauma patient.

## Key Findings
- Patient averages: OD 277.7 µm, OS 276.3 µm (Normal)
- 3 sectors flagged "Edema Risk" (315, 304, 313 µm)
- Rule-based classifier using medical thresholds

## Tech Stack
Pandas -  Matplotlib -  Seaborn -  Plotly -  Streamlit
